# Exo Feature Roadmap

## Phase 1
- workspace-centric IDE shell
- markdown notebook editor
- plain terminals
- deterministic visual validation

## Phase 2
- knowledge-system parity:
  - backlinks
  - tags
  - search
  - branches
  - workspace-aware navigation

## Phase 3
- runtime and memory:
  - durable memory
  - trace archive
  - retrieval/index
  - working-memory assembly
  - CLI-first memory operations

## Phase 4
- research harness:
  - workcells
  - supervised runs
  - metrics
  - artifacts
  - logs
  - `autoresearch-macos`

## Phase 5
- multi-agent system:
  - multiple terminals
  - terminal grids
  - subagent coordination
  - agent operator views

## Phase 6
- datasets, evals, and training:
  - objective contracts
  - dataset assignment
  - eval suites
  - operator labels
  - local helper-model training
