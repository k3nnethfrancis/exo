# Plugins Are Distribution Bundles, Not Runtime Seams

Exo reserves **Plugin** for a future installable, versioned bundle used to distribute proven Skills, ontology templates, Command templates, evals, or external capability configuration. Skills author behavior, Commands and Providers execute or supply capabilities, and each retains its own authority boundary. This rejects arbitrary in-process renderer extensions and prevents a future marketplace concern from dictating today's core interfaces.

This ADR does not authorize a Plugin Manager, manifest runtime, profile system, marketplace, or plugin-owned settings surface.
