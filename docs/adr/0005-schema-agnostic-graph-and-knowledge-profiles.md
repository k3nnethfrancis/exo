# Preserve graph facts and interpret them without closing the vocabulary

Exo's canonical knowledge remains user-owned Markdown and frontmatter. Its
derived Knowledge Graph preserves arbitrary Properties, open Concept Types,
Relations, resolution, origin, and Evidence without imposing a universal
ontology. Note Root Formats project base Concepts; an optional reviewed
Workspace Ontology may interpret selected Properties and relationship rules.
Generic Markdown is the zero-configuration Format and Open Knowledge Format 0.1
is the first interoperability Format.
Graph Views and quality evals consume this model but cannot silently promote
inferred similarity into document-origin knowledge or reject unknown user data. This
extends Folder Indexes as the first ontology substrate without creating an
app-owned schema database.
