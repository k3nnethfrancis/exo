# Preserve graph facts and interpret them through optional Knowledge Profiles

Exo's canonical knowledge remains user-owned Markdown and frontmatter. Its
derived Knowledge Graph will preserve arbitrary properties, open concept types,
relations, resolution, authority, and evidence without imposing a universal
ontology. Optional user-owned Knowledge Profiles may interpret selected
properties and relationship rules; Generic Markdown is the zero-configuration
profile and Open Knowledge Format 0.1 is the first interoperability profile.
Graph Views and quality evals consume this model but cannot silently promote
derived similarity into authored knowledge or reject unknown user data. This
extends Folder Indexes as the first ontology substrate without creating an
app-owned schema database.
